HTML이란 무엇인가?

HyperText Markup Language(HTML) is The standard markup Language for Creating web pages and web application. Web browsers receive HTML Documents from a Web Server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page senmatically and originally included cues for the apprearance of the document.

HTML elements are the building blocks of HTML Pages. with HTML Constructs, images and other objects, such as interactive forms, may be embedded into rendered page. it provides a means to create structured documents by denoting structural semantics for text such as headings,paragraphs,lists,links,quotes and other items.
